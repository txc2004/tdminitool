TD - MiniTool v0.5.1

Top Menu - Buildings List - Mini Map - Bottom Menu - Bars Graph


TD - MiniTool is a window application, you can have it together with game to see what the weapons output, like: Efficiency, Hits, DamagePerFire, DamagePerSecond, DamageUsed, DamageWasted, CostEfficiency after each wave. You can folow the progress of the weapons each wave with bars graph. You can also easily setup a game for tests and restore waves.

Download : https://drive.google.com/drive/folders/1AGb7TPTBk7GUSmtbF4-aY4gLBEcfc3Ts 
 

- Top Menu -

Menu

  • Open: You can open any Progress.gd files, anywhere located on PC for view or restore it. By default app open the Progress.gd logated on game folder.
     
  • Get Data: get the data from the Open File, decoded and set on clipboard, to pasted them every you like to edit them.
     
  • Set Data: When you edit and copy them, then take it from clipboard recode and return it on Open File.
     
  • Hack: If you have to do tests or Play for Fun, it edit the Open File and add a plenty Supply/Recon/Tech/Health also give x35 each building you have on inventory, also add Basic Weapons or All Weapons.
     
  • Opacity: Can you set the opacity for the app from 60 to 100 by steps of 5. Default is 95.
     
  • Auto Hide: The app hide on side when click out of Window, appear when you click on Window.
     
  • Restore: You can go back on previous waves or restore any Progress.gd you have already. It clear the waves when you start a new game and place the first tile.
     
  • Help: It open the Help file.
     

Important! Game must closed before use Set Data/Hack/Restore. Also always make first a Back Up your original Progress.gd.


Efficiency/Hits

  • Eff: You can see the efficiency of Weapons and measured by: (100 / (DamageUsed + DamageWasted)) * DamageUsed.

     

Buildings with multiply weapons calculated separately, summed and then per number of weapons.

 

  • Hts: Show the total hits a weapon reach. Examples: If a rifle do 10 shoot on 1 target count as 10 hits. If a splash weapon hit with one bomb 10 targets is again 10 hits.

     

You can switch between Eff/Hts by click on them.


DamagePerFire/DamagePerFireMax

  • Dpf: DamagePerFire of each single fire a weapon do and calculated by: (DamageUsed + DamageWasted) / HitsCount

     

  • DPF: DamagePerFire include Max Targets and calculated by: ((DamageUsed + DamageWasted) / HitsCount) x MaxTargets

 

You can switch between Dpf/DPF by click on them.


DamagePerSecond/DamagePerSecondMax

  • Dps: DamagePerSecond. Calculated by: (Dpf * magazineSize) / ((magazineSize / rateOfFire) + reloadTime)

     

  • DPS: DamagePerSecond include Max Targets and calculated by: (DPF * magazineSize) / ((magazineSize / rateOfFire) + reloadTime)

 

You can switch between Dps/DPS by click on them.


DamageUsed/DamageTotal

  • Dmg:  DamageUsed by weapon without the DamageWasted.
     
  • DMG: Total damage of a weapon, DamageUsed + DamageWasted.
     

You can switch between Dmg/DMG by click on them.

For Hts/Dps/DPS/Dpf/DPF/Dmg/DMG Buildings with multiply weapons, calculated separately and summed.


Cost/Efficiency

  • Cst/Eff - Dpf/DPF: CostEfficiency / DamagePerFire, CostEfficiency / DamagePerFireMax
     
  • Cst/Eff - Dps/DPS: CostEfficiency / DamagePerSecond, CostEfficiency / DamagePerSecondMax
     
  • Cst/Eff - Dmg/DMG: CostEfficiency / DamageUsed, CostEfficiency / DamageTotal
     

The type of Cst/Eff you can choose it on bottom menu.

The Game count: 1 Recon = 2 Supply, 1 Tech = 250 Supply. The Cst/Eff count that also.


Sort

Sort the Buildings list by:

  • Default: Place order.
  • Name: Buildings Name from A to Z.
  • Hts/Dps/DPS/Dpf/DPF/Dmg/DMG/Cst/CST: From higher to lower.
     

You can also quick sort the the items with right mouse button.


Quick Hide

  • <: It hide the app when you click it.

- Buildings List -

On list the first column is the header. For Eff show the average and for Hts/Dps/DPS/Dpf/DPF/Dmg/DMG/Cst/CST show the total.

The items after headers is:

  • Infantry: All the Infantry units, Heroes, Grenadiers, Chaplain, Medic.
  • Armory: Tanks.
  • HQDamage: The Damage take the HQ from foes.
  • Blue/Red/Green HQ: HQ weapons, if has.

 

If you click on item, can you see on minimap the tile where located.

You can increase the list height and width, by resize the window.

- Mini Map -

  • If you click on an tile, you can see on list the buildings is on that tile.
  • You can resize the minimap area, on that color spot between list and minimap.
  • You can drag the minimap with right mouse button.

 

- Bottom Menu -

  • The Basic/Shield/Air show you the damage the weapons can do, as the game shows on GUI if you have enable them on games settings menu.
  • The InGame show you the damage the weapons actual down after the wave end.
  • The middle item, it choose the Cost/Eff type.
